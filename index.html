Resumo Desenvolvimento Web 1 - HTML

1. Conceitos Básicos

WWW: Sistema de informação com URLs (localização), HTTP (protocolo) e HTML (linguagem).

URL: Uniform Resource Locator, used to identify resources on the web.

HTTP: Hypertext Transfer Protocol, used for communication between client and server.

HTML: Hypertext Markup Language, used to create web pages.

Internet: Rede global de computadores interconectados que permite a troca de informações.

Serviços Internos da Internet: E-mail, FTP, VoIP, VPN, entre outros, que utilizam a infraestrutura da internet para comunicação e transferência de dados.

Web: Serviço que utiliza a internet para acessar documentos e recursos através de navegadores.

Estrutura Básica:

HTML: Linguagem de marcação com tags (<tag>conteúdo</tag>). 

<!DOCTYPE html>
<html>
<head>
    <title>Título</title>
    <meta charset="UTF-8">
</head>
<body>
    Conteúdo
</body>
</html>
2. Elementos HTML
Elemento Descrição Exemplo
Títulos <h1> a <h6> <h1>Título</h1>
Parágrafos <p> <p>Texto</p>
Links <a href="url"> <a href="pagina.html">Link</a>
Imagens <img src="img.jpg" alt="texto"> <img src="foto.jpg" alt="Foto">
Listas <ul>, <ol>, <li> <ul><li>Item</li></ul>

3. Formulários

Estrutura: <form action="url" method="get|post">

Tipo Descrição Exemplo
Texto <input type="text"> <input type="text" name="nome">
Senha <input type="password"> <input type="password" name="senha">
Rádio <input type="radio"> <input type="radio" name="sexo" value="M">
Checkbox <input type="checkbox"> <input type="checkbox" name="opcao" value="1">
Select <select> com <option> <select name="estado"><option value="RS">RS</option></select>
Textarea <textarea> <textarea name="msg" rows="4"></textarea>
Botões <input type="submit|reset|button"> <input type="submit" value="Enviar">

Atributos: name, value, placeholder, required, readonly, disabled, checked, selected

4. Tabelas

<table border="1">
    <tr>
        <th>Cabeçalho</th>
        <td>Dado</td>
    </tr>
</table>

Atributos: colspan (mesclar colunas), rowspan (mesclar linhas)

5. Multimídia

Vídeo:

<video width="320" controls>
    <source src="video.mp4" type="video/mp4">
</video>

Áudio:

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
</audio>

6. HTTP

Métodos: GET (dados na URL), POST (dados no corpo)

Códigos: 200 (OK), 404 (Não encontrado), 500 (Erro servidor)

HTTPS: Versão segura com criptografia

7. Navegação

Links:

  • Mesma pasta: <a href="pagina.html">
  • Pasta abaixo: <a href="pasta/pagina.html">
  • Pasta acima: <a href="../pagina.html">
  • Raiz: <a href="/pasta/pagina.html">

8. Outros

Favicon: <link rel="icon" href="favicon.ico">

iframe: <iframe src="url"></iframe>

Caracteres especiais: &nbsp; (espaço), &lt; (<), &gt; (>)
